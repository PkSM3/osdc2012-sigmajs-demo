Explore Reddit comments network with sigma.js


The graph bellow represents Reddit users who commented the selected Web page. Two user are connected when at least one of them replied to the other.

The thickness of an edge is proportional to the number of replies between the two connected users. The size of a user is related to the number of comments he wrote about the selected page.

You can navigate through the graph with the arrow keys to move across the graph, +, - to zoom in and out, and the spacebar to reset the graph position.
